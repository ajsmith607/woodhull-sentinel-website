# Woodhull Sentinel Digital Newspaper Archive Website 

The Town of Woodhull in Steuben County New York owns a physical collection of newspapers digitized by Backstage Library Works in Bethlehem PA in 2025. This repository contains code for a JAMStack site to provide basic search, browse, and page views of the digitized collection.

## Technical Details

- Website built with [Eleventy](https://www.11ty.dev/) static site generator.
- Search powered by [FlexSearch](https://github.com/nextapps-de/flexsearch).
- Image viewer supports pan and zoom via [Panzoom](https://github.com/timmywil/panzoom).
